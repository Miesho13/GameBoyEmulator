*** GAME BOY EMUlATOR ***


This repository contains a project that emulates the Game BoyTM. 
This emulator aims to provide a faithful recreation of the 
original Game BoyTM gaming experience.

Features

- Accurate emulation of Game BoyTM hardware
- Support for original Game BoyTM game cartridges (ROMs)
